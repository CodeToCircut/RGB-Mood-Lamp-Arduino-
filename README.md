# RGB-Mood-Lamp-Arduino-
Build your own custom RGB mood lamp using an Arduino!

In this project, I walk through how to wire up an RGB LED and use some simple code to cycle through colors, creating the perfect ambient lighting for your desk.

🛠 Components Used

Microcontroller: Arduino Uno

Light Source: RGB LED (Common Cathode/Anode)

Input: 10kΩ Potentiometer

Resistors: 220Ω (x3)

Hardware: Breadboard and Jumper Wires
